Core algorithm
1. coord_set = set(map(tuple, coords))
2. count = 0
3. for p1 in coords:
4.   x1, y1 = p1
5.   for p2 in coords:
6.     x2, y2 = p2
7.     if x2 <= x1 or y2 <= y1: continue
8.     Compute TL + BR, check set
Corner check
top_left = (x1, y2)
bot_right = (x2, y1)
if top_left in coord_set and \
   bot_right in coord_set:
     count += 1
